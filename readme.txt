@par Example Description 

This example provides a means of controlling an LG window AC via duplicating their 
variant of Manchester encoding protocol. I reverse engineered the protocol using 
a logic analyzer.
        
        
@par Hardware and Software environment  

  - This example runs on the STM32F4xxx
    
  - I wrote this for use on the STMicroelectronics STM32F429I-Discovery RevB
